Hi, I'm Cuong — a software engineer based in Vietnam. I spend most of my time writing Go, occasionally Rust, and building tools that help me (and sometimes others) work more effectively.

This blog is where I think out loud. I write about things I'm working through: distributed systems, clean architecture, developer tooling, and the occasional detour into how coding agents fit into day-to-day engineering.

I work mostly in the backend, but I care deeply about the full stack of the experience — from API design to the command-line interface that wraps it.

## What I work on

- **Go services** — microservices, worker pools, ETL pipelines, API design
- **Developer tooling** — shell scripts, CLI tools, productivity automation
- **Coding agents** — experimenting with Claude Code and parallel agent workflows
- **Open source** — small, focused projects on [GitHub](https://github.com/cuongtranba)
